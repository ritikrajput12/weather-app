# Weather App 🌤️

A weather app built with React JS that shows real-time weather data.

## Features
- Search weather by city name
- Shows temperature, humidity, and wind speed
- Dynamic weather icons
- Responsive design for mobile

## Tech Stack
- React JS
- Vite
- OpenWeatherMap API
- CSS

## Live Demo
[Click here](https://weather-app-six-black-88.vercel.app)

## How to Run Locally
1. Clone the repo
2. Run `npm install`
3. Add `.env` file with `VITE_APP_ID=your_api_key`
4. Run `npm run dev`
