# Weather App

This is a React-based weather application that fetches real-time weather data using an external API.
Users can search for any city and view key weather details instantly.

Live Demo: https://weather-app-six-black-88.vercel.app

---

## Overview

This project was built to practice working with APIs, handling asynchronous data, and displaying dynamic UI based on real-time responses.

---

## Features

* Search weather by city name
* Displays temperature, humidity, and wind speed
* Dynamic weather icons based on conditions
* Responsive design

---

## Tech Stack

* React
* Vite
* OpenWeatherMap API
* CSS

## Setup

Clone the repository:

```bash id="t3sh7f"
git clone https://github.com/your-username/weather-app.git
cd weather-app
```

Install dependencies:

```bash id="zoyp9t"
npm install
```

Create a `.env` file and add:

```env id="3r9t7p"
VITE_APP_ID=your_api_key
```

Run the app:

```bash id="j1x8qf"
npm run dev
```
